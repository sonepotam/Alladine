Внимание. На компьютере должен быть установлен 32-разрядный клиент Oracle. 
Для его установки необходимо обратиться в отдел сопровождения.




Правильность работы программы гарантируется только при использовании в качестве
обозревателя интернета Microsoft Internet Explorer версии 5 и выше. 
Войдите в меню Сервис -> Свойства обозревателя -> Безопасность. 
Для местной интрасети необходимо разрешить загрузку, запуск и использование 
элементов ActiveX.
